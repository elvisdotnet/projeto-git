<!-- Readme -->
Aprendendo e treinando alguns comandos git.

----

git --version --> Versão do Git instalada.

----

git init --> Inicia um repositório git vazio.
Initialized empty Git repository in ...

----

git add --> Adiciona a "staging area" ("área de preparação/espera para o commit") os arquivos modificados e novos, pode ser utilizado mais de uma vez.

git add arquivo.extensao (git add readme.md)

-----

git status --> Mostra as mudanças a serem commitadas e se há arquivos modificados para adicionar.
Chages to be committed:
new file: mostra o arquivo

----

git commit -m --> Registra as alterações no controle de versão, é como uma "fotografia" do código no momento.

git commit -m "texto do commit"

----
